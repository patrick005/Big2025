    printf("MINUS %d - %d = %d\n", a, b, a-b);

    printf("DIVISION %d / %d = %d\n", a, b, a/b);

}
연산자
1) 산술연산자 : +, -, *, /, %
2) 대입연산자: =
3) 복합연산자: (1),2)의 결합) +=, -=, *=, =+
4) 증감연산자: ++, --
	- 전위: ++a -> 올려서 출력해라
	- 후위: a-- -> 출력하고 올려서 저장해라
