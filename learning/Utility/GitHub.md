# Git 연동  

1) 터미널에서 git관련 기능 설치  
  ```$sudo apt-get install git```  
2) Git초기화 및 로컬 저장소 생성 (처음 설정시 적용)  
  ```$git init```  
3) Git 링크와의 연결  
  ```$git remote add origin [Git링크]```  
4) .git이란 로컬 저장소가 생겼는지 확인  
  ```$ls -al```  
5) GitHub에서 최신 변경사항 가져오기  
  ```$git fetch origin```
